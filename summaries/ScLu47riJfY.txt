# Summary

This video discusses the growing financial unsustainability of AI coding tools, using Microsoft and Uber as case studies. Microsoft has cancelled Cloud Code licenses for thousands of developers, forcing them to use GitHub Copilot CLI instead due to cost savings. Uber serves as a cautionary tale, having exhausted its entire 2026 AI budget in just four months using Cloud Code, with some engineers spending up to $2,000 for single sessions. The core problem is the shift from per-seat pricing to token-based pricing models, which creates unpredictable escalating costs as usage increases. The speaker emphasizes a critical gap: while companies celebrate faster development, nobody discusses whether these massive expenditures generate actual return on investment or increased revenue. The analysis reveals that AI pricing is rising across providers—from OpenAI to newer models—contradicting industry assumptions that costs would decrease. Chinese alternatives like DeepSeek offer significantly cheaper pricing with temporary discounts, though costs there are also climbing. The video targets developers and tech professionals concerned about AI tool economics, highlighting how the current pricing model creates an unsustainable "all-you-can-eat token buffet" that major corporations are now actively curtailing.
