# Video Summary

This video provides a comprehensive performance comparison between Apple's M3 Ultra Mac Studio and Nvidia's RTX 5090 graphics card for running large language models using tools like Ollama and LM Studio. The creator tests various models ranging from Gemma 3 (4 billion parameters) to DeepSeek R1 Distill Llama (70 billion parameters), measuring token generation speeds, power consumption, and memory utilization on both systems. Key findings reveal that for smaller models fitting entirely in the RTX 5090's 32GB VRAM, both machines perform comparably with the RTX 5090 slightly ahead, but power consumption differs dramatically (356W vs 113W). However, for larger models exceeding VRAM capacity, the Mac's 512GB unified memory architecture proves superior—the RTX 5090 bottlenecks severely when offloading to system RAM (dropping to 1.8 tokens per second), while the Mac maintains usable performance at 13 tokens per second. The video targets machine learning practitioners and AI developers considering hardware investments, demonstrating that the Mac's unified memory architecture provides substantially better value for running large models despite higher upfront costs, making it ideal for serious AI workloads even if per-token performance on smaller tasks is comparable.
