This stream features a developer reviewing web portfolios and projects submitted by participants during a live event. The host conducts evaluations using Cloud Fable 5, an AI-powered code analysis tool that assesses websites across multiple criteria including design, web performance, UX/accessibility, SEO, semantic HTML, security, and responsive functionality. Each project receives scores from both Cloud (0-100 scale) and the host, with detailed constructive feedback highlighting common mistakes like mixing languages, poor semantic structure lacking proper H1 tags, generic alt text, outdated copyright years, missing introductory sections, and incorrect chronological ordering of work experience. The host emphasizes practical web development best practices such as maintaining consistent language, prioritizing recent professional experience at the top, ensuring proper heading hierarchy for SEO, and avoiding AI-generated generic descriptions. Notable projects reviewed include a portfolio with interesting animations but lacking introduction, and another featuring an AI chatbot assistant. The stream is interrupted repeatedly by internet connectivity issues, which the host uses as a teaching moment about handling mistakes gracefully. The content targets junior developers and portfolio creators seeking to improve their web projects through professional critique.
