# Summary

The creator undertook a month-long project to improve their notoriously poor memory by learning competitive memory techniques and attempting to memorize 3,141 digits of pi. After consulting with Morgan Berens, a memory researcher, and Nelson Dulles, a five-time USA memory champion, they learned that effective memory relies on identifying unique retrieval cues and employing the ancient technique of memory palaces—mentally placing encoded information within familiar physical spaces. Using the Person-Action-Object (PAO) system combined with phonetic code conversion, they encoded pi into memorable images and distributed them across eleven interconnected memory palaces derived from locations like childhood homes and grocery stores. Over several weeks, the creator successfully memorized thousands of digits and eventually rented out a theater to publicly recite their achievement in front of an audience. Beyond the novelty accomplishment, the project functioned as a deeper exploration of memory mechanics and how intentional attention to unique details—rather than trying to capture everything—enables lasting recall. The video targets viewers interested in memory science, self-improvement through unconventional methods, and the documentary-style journey of accomplishing an ambitious personal goal.
