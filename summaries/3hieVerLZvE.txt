Anthropic released Claude Opus 4.8 on May 28th, positioning it as their most powerful publicly available model with improvements across all benchmarks while maintaining the same pricing as Opus 4.7. The update features enhanced honesty where the model is four times more likely to acknowledge when it cannot solve a problem, addressing a common AI issue of overconfident hallucinations. New features include controllable effort levels on Claude.ai and Dynamic Workflows in Claude Code for enterprise users, which can spawn hundreds of parallel sub-agents—though this sparked controversy for triggering automatically when the word "workflow" appeared in prompts. A fast mode enables 2.5x speed at double the token cost, though still significantly cheaper than previous expensive iterations. Pricing remains competitive at $3-25 per million tokens for input/output. The video creator, who built a local AI subtitle generation application using Opus 4.8 with Whisper and WebGPU, expresses mixed impressions, noting the model performs adequately but lacks a substantial qualitative leap from 4.7 and sometimes struggles with comprehension, though he praises Cursor's superior development experience compared to Visual Studio Code integration.
