# Video Summary

This video presents 38 curated web design inspiration resources designed to help designers build award-winning websites. The presenter covers a diverse range of platforms serving different design needs and specializations. Featured resources include Chrome extensions like Muzli for daily inspiration browsing, landing page-specific sites like Landing Pics and Landings, comprehensive curators such as Awwwards and CSS Design Awards with voting mechanisms, component-focused tools like Unsection and Landbook that allow searching by specific design elements, and specialty platforms including Commerce Cream for e-commerce design and Brutalist Websites for experimental layouts. The list also encompasses design portfolio platforms like Dribbble and Behance, video-based inspiration from Godly, interaction design resources like Hover States, and supplementary sources covering typography (Fonts in Use), branding (Brand New), and code tutorials (Codrops). The content targets web designers seeking diverse visual references across different platforms, industries, and design approaches to avoid homogeneous work and continuously improve their skills and creativity.
