This talk covers Cloud Code, an AI-powered coding agent built on Claude that functions as a terminal-savvy co-worker. The speaker explains how Cloud Code works as a pure agent with tools for file editing, terminal commands, and codebase exploration through agentic search—using familiar commands like grep and find rather than complex indexing systems. Key use cases include onboarding to new codebases, discovering code structure, implementing features with human guidance, zero-to-one project building, and large-scale migrations. The presentation details best practices such as using claude.md files for persistent instructions, managing permissions efficiently, integrating CLI tools and MCP servers, and monitoring context window usage with slash-clear and slash-compact commands. Advanced techniques include test-driven development workflows, using screenshots for multimodal guidance, running multiple Claude instances simultaneously, and leveraging the escape key for course correction. The speaker also highlights newer features like model switching and emphasizes that Cloud Code excels at terminal operations, making it particularly valuable for git workflows, Docker commands, and other CLI-based tasks. The tool targets developers across experience levels seeking to accelerate coding workflows.
