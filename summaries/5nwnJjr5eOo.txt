This video explores Aerospace, a tiling window manager for macOS inspired by i3 Linux window managers, as a solution for improving productivity on Apple computers. The creator compares macOS customization to coffee appreciation—most users accept defaults, while power users seek optimization. Aerospace enables efficient window management through keyboard-driven tiling, workspace switching, and customizable layouts similar to tmux in the terminal. Key features include Vim-key navigation (alt+hjkl), window resizing, accordion and vertical/horizontal layout modes, floating windows, and a tree-based window hierarchy that supports nested groupings. The video demonstrates practical workflows like fuzzy-searching open windows via fzf, configuring application launchers that replace skd, and using service modes for advanced customization. Additional tools discussed include Sketchy Bar for visual workspace indicators and Junky Borders for window focus visualization. The content targets macOS power users, developers, and terminal enthusiasts seeking deeper system control and keyboard-centric workflows, drawing parallels to tmux mastery. Configuration is YAML-based and stored in ~/.config/Aerospace, making it approachable for technical users familiar with dotfile management.
