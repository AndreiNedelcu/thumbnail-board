# Thumbnail Board

Visual reference board for YouTube thumbnails, organized by style tags from thumbnailexamples.com.

**Live site:** https://andreinedelcu.github.io/thumbnail-board

## Features
- 770 thumbnails with official tags
- Real-time search by tag, title or channel
- Filter by category (STYLE, MOOD, TEXT, ELEMENT…)
- Lightbox with full-size image, all tags, YouTube/Eagle/copy actions
- Click any tag in lightbox to filter by it
